📋 설정 방법

  1. 텔레그램에서 @BotFather를 검색하고 /newbot 명령어로 봇 생성
  2. 발급받은 Bot Token을 위 입력란에 붙여넣기
  3. "연결" 버튼 클릭
  4. 생성한 봇의 대화창에서 /start 전송 → Chat ID 자동 등록
  5. "자동 연결" 체크 시 다음 접속부터 자동으로 연결됩니다

🎮 봇 명령어

/start봇 연결 및 Chat ID 자동 등록
/reserve텔레그램에서 열차 검색 및 예약 시작
예: /reserve 수서 부산 2026-03-01 06:00
/stop실행 중인 매크로 원격 중단
/restart최신 데이터로 매크로 재시작
/status현재 매크로 상태 상세 확인
/trains마지막 검색된 열차 목록 보기
/chatid현재 Chat ID 확인
/help명령어 목록 표시

/reserve 수서 부산 2026-03-01 06:00
